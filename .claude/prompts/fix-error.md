# Fix Error Prompt

Use this when you get an error:

---

I got this error:

```
[paste the error message here]
```

Please:
1. Tell me what this error means in plain English
2. Tell me exactly which file and line to look at
3. Show me the broken code and the fixed code side by side
4. Explain why the fix works

Don't just fix it - help me understand so I can learn.
