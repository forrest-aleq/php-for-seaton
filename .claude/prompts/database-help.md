# Database Help Prompt

Use this when working with the database:

---

I need to [save / get / update / delete] data in the database.

The data is: [describe what data]

Please:
1. Use PDO (not mysqli or mysql_)
2. Use prepared statements with :named parameters
3. Show me the complete code
4. Explain what each line does

Remember: NEVER put variables directly in SQL strings.
